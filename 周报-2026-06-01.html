多 Agent 协作架构兴起

单一 Agent 向多 Agent 协作演进，出现 Planner-Agent-Executor 三层架构。Planner 负责任务分解，多个 Specialist Agent 并行执行，Executor 整合结果。Salesforce 和 Databricks 已率先落地。

Data Agent 从被动响应用户查询，转向主动发现数据异常、趋势和机会点。阿里云小Q洞察、观远智能归因等产品已实现自动化根因分析和预警推送。

企业级能力成为竞争焦点
