Section 02

国内厂商动态

字节跳动 · 火山引擎 DataWind
稳步推进

产品现状：DataWind 深度集成豆包大模型，提供 ChatBI 问数、自动归因、趋势预测、报告生成等 AI 能力。技术架构已演进至 3.0 版本——数据探查 Agent + 数据洞察 Agent + 上下文引擎 + Agent Workspace 四模块协作。

  • 2026年3月行业评测中综合排名第二，"通用大模型能力突出，适合快速验证和轻量级应用"
  • 关键洞察：单 Agent 有效工具上限约 15 个，超出需转向 Supervisor 多 Agent 模式
  • 数据智能业务独立成线，强化与豆包 AI 协同

商业化：云订阅制为主，Data Agent 以增值模块叠加销售，主攻互联网/电商/消费品客群。

来源：IT之家评测 · 火山引擎官网 · 架构实录
阿里云 · Quick BI / 智能小Q
全面升级

产品现状：连续 6 年入选 Gartner ABI 魔力象限（中国唯一），核心战略升级为 5 大 Agent 驱动——报告/问数/解读/洞察/搭建。

  • 企业知识库升级，支持直连第三方知识库/本地文件
  • 报告 Agent 升级：编辑时可直接唤起多维度归因
  • 通过信通院"AI Agent 智能体产业图谱"专项评测（首批）
  • 推出 Quick BI + RDS DuckDB 免费组合包引流

商业化：四档年订阅，AI Agent 以高阶版本/增值包销售，付费小Q客户超百家。

来源：阿里云 Quick BI · 小Q报告文档
帆软 · FineBI / FineChatBI
持续迭代

产品现状：国内 BI 市场份额长期领先，战略定位 "AI For BI"。核心产品线：FineChatBI（对话式取数）、FineBI 内嵌 FineAI 助手（V3.24.0）、FineReport（AI 辅助报表设计）、FineDataLink（数据集成底座）。

  • FineAI 助手支持接入 DeepSeek/通义千问等 OpenAI 标准接口
  • 研发资源向 AI 倾斜，FineChatBI 作为下一代核心产品持续迭代

商业化：FineReport 永久授权+年维护；FineBI 免费个人版+企业版订阅。核心竞争力在私有化部署，政企/金融/制造渗透率高，非上市，外界估值独角兽级。

来源：FineBI AI 助手文档 · 帆软趋势分析
观远数据 · 观远Agent
三品齐发

产品现状：消费品/零售行业 BI 标杆厂商，2024-2025 年推出问数 Agent、洞察 Agent、DataFlow 三款新品。

  • 持续强化"问数 → 洞察 → 决策"闭环
  • 推进全面信创适配，产品自主可控

商业化：SaaS 订阅+私有化双模，已完成 D 轮融资（红杉中国、高榕资本等）。

来源：观远数据官网 · 产品博客
亿问 · Data Agent
深耕场景

产品现状：独创 NL2LF2SQL 双引擎架构——逻辑引擎（图计算+符号逻辑，GPU=0 Token=0）+ 语义引擎（Transformer），Token 消耗降低 65%，语义定位准确率 99.8%。

典型客户：麦当劳、蜜雪冰城、康师傅、理想汽车、华为、银联商务等。

商业化：企业定制化方案为主，偏中大型企业客户。

来源：亿问数据官网
思迈特 · SmartBI 白泽
评测第一

产品现状：定位"智能体数据决策分析平台"，多智能体协同 + 可视化工作流编排 + RAG知识增强 + ReAct推理 + MCP/A2A协议兼容 + 指标模型引擎（准确率 ≥98%）。

  • 2026年1月：白泽入选《中国 AI 智能体百强》，唯一原生 BI 厂商
  • 2026年3月：IT之家评测第一，IDC 7项平台能力全部第一，金融/央国企满分
  • 累计 80+ 软著、近 20 项发明专利，发明专利数 BI 行业第一

商业化：私有化部署高客单价（金融/央国企），平台授权+实施+年维护，深耕 15 年。

来源：SmartBI 官网 · AI百强入选 · IT之家评测

  • FY26 Agentic + Informatica ARR 达 $2.9B；Data 360 摄入量 112T+ 行

来源：Tableau 定价 · Futurum 分析 · Agentic Analytics
Databricks · Genie Code / DIP
重磅发布

产品现状：以技术深度和开放架构取胜。

  • 开源策略：Unity Catalog 开放治理基座，Lakeflow Connect 免费层 100 DBU/天
来源：Genie Code 发布 · FabCon 2026 · Verdantix 分析

Section 05

产品研究与趋势

技术架构趋势

Text-to-SQL：从"能用"到"可信"

当前 SOTA 在 BIRD 榜单达 71.83%，但 BIRD 存在 52.8% 标注错误，Spider 2.0 真实场景 GPT-5 准确率不足 20%。

破局手段原理代表
Schema Linking图谱化字段映射，解决晦涩命名中国电信星海智文（BIRD Top 4）
Agentic Loop生成 → 沙箱试运行 → 自反思修正火山引擎 3.0、SmartBI 白泽
NL2MQL2SQL插入指标语义层，从源头降低幻觉Aloudata、Tableau、GoodData

多 Agent 协作进入 3.0 阶段

阶段架构局限
1.0Plan-Execute 静态灵活性不足
2.0One Agent 动态工具上限约 15 个
3.0（当前）Supervisor + 专业 Worker Agent需要精细编排

产品发展趋势

两大行业共识

1. Governance-first：Salesforce 和 Databricks 均将数据治理作为 Agent 时代核心基础设施。微软调研显示 29% 员工已在使用未授权 AI Agent，"影子 AI"成最大治理黑洞。

2. Semantic Layer 升格为战略资产：AI Agent 与传统 BI 的融合点在语义层，拥有高质量语义层和业务 Know-How 的厂商获超额溢价。

演进阶段时间特征代表
Chat BI2022-2023NL2SQL 包装，用户主动驱动ThoughtSpot 早期
增强式 Agent（主流）2024-2025多轮对话+异常检测+报告生成SmartBI 白泽、火山引擎
自主式 Agent（探索）2026→2027+自主计划、执行代码、触发业务动作Databricks Genie Code

三大可能技术突破

1
数据专域模型
通过真实场景 Agentic Training 训练的专域模型将显著超越通用 LLM
2
MoE 轻量化降成本
Qwen3-Next 等 MoE 模型推理吞吐比稠密模型快 10 倍+，可部署到企业私有环境
3
A2A/MCP 协议标准化
不同厂商 Data Agent 可互操作，BI 平台从"全栈产品"演变为"Agent 生态市场"
中国市场数据不出境 + 国产化政策为本土厂商创造 2026-2028 年独特竞争窗口期。BI 工具用户渗透率将从 15-20% 跃升至 60%+。竞争逻辑从"工具功能"转向"所能产生的智能质量"。
